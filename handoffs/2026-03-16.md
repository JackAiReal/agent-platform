## 常用命令

### 后端
```bash
cd maixu-system
npm run prisma:generate
npm run prisma:migrate
npm run prisma:seed
npm run dev:server
```

### 前端 H5
```bash
cd maixu-system
npm run dev:client:h5
```

### 前端小程序
```bash
cd maixu-system
npm run dev:client:weapp
```

### 生产构建检查
```bash
cd maixu-system
npm run build:server
npm run build:sdk
npm run build:client:h5
npm run build:client:weapp
```
