Agent Management Platform

Track diary/output/cost, review records, run schedule jobs, and push new entries to Telegram/X webhook pipelines.

Open Console OpenClaw

Features

Diary, output, cost modules with review status and trend charts.

Schedules

Built-in schedule list backed by OpenClaw cron jobs.

Webhooks

Auto push for newly created entries to Telegram and custom X relay endpoint.
